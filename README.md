# Online Retail Customer Segmentation

## Project Overview

This project applies Unsupervised Machine Learning techniques to segment customers based on purchasing behavior using an online retail dataset from Myntra Gifts Ltd.

The objective is to identify meaningful customer groups and provide actionable business recommendations.

---

## Business Problem

Myntra Gifts Ltd. wants to understand customer purchasing behavior and identify:

- High-value customers
- Dormant customers
- Growth-potential customers
- Occasional buyers

Customer segmentation helps improve:

- Marketing effectiveness
- Customer retention
- Inventory planning
- Revenue generation

---

## Dataset Information

Source: Online Retail Dataset

Period:
December 2010 – December 2011

Records:
541,909 transactions

Features:

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

## Key Results

- 4,338 customers analyzed
- 4 customer segments identified
- K-Means selected with K=4
- Silhouette Score: 0.281

### Customer Segments

1. Moderate Customers
2. Dormant Customers
3. High-Value Loyal Customers
4. Occasional Premium Customers
